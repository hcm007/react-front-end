# react-front-end

### Installing

```
git clone git@github.com:hcm007/react-front-end.git
```
```
cd react-front-end
```
```
npm install
```

### Running
Build the new bundle.js
```
npm run build
```
Watch the new index.js 
```
npm run dev
```
